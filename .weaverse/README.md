# .weaverse

This folder contains **Weaverse org-internal** resources used for development and planning of the Pilot theme.

## Contents

- `specs/` — Feature specifications following the [Spec-Driven Development](../AGENTS.md#spec-driven-development-sdd) conventions. Each feature has its own subfolder with a README, implementation plan, and optional work logs.
- `autoperf/` — Autonomous performance optimization tool that uses Claude Code CLI to iteratively improve Lighthouse scores via a build-measure-decide loop.
- `customer-account-local-dev.md` — Guide for running Hydrogen locally with Customer Account API support.

## For template users

This folder is safe to delete. It contains internal planning documents and tooling specific to the Weaverse team and is not required for your storefront to function.
