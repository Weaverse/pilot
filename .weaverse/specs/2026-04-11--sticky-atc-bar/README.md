## Original Prompt

> Look at the @app/sections/main-product/ please, when we scroll the atc button out of the viewport, we need a sticky atc button element show up, how should we built that?

## Summary

When users scroll past the Add-to-Cart button on the product page, a fixed bottom bar appears with the product image, title, price, and an ATC button. This provides persistent purchase access without scrolling back up, especially important on mobile.
