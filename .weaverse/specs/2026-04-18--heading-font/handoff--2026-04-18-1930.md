  1. `npm uninstall @fontsource-variable/playfair-display`
  2. `npm install @fontsource-variable/[new-font]`
  3. Update import in `app/root.tsx`
  4. Update type declaration in `app/types/fontsource.d.ts`
  5. Update CSS variable in `app/styles/theme.css`
  6. No changes needed to component/route files - they all use semantic h1-h6 tags
